Welcome to the Findex Ferry Terminal!

The Findex Ferry Terminal is the latest and greatest initiative for Findex Digital

Findex has bought a small ferry terminal and two ferries are available at any one time.

Ferry Types:

Small Ferry
A Small Ferry can hold 8 small vehicles
Large Ferry
A Large Ferry can hold 6 Large vehicles

The Test

There is an API described by the interface IVehicleProvider which currently has one implementation, HardcodedVehicleService, which provides a random vehicle at the click of the button. The goal for this test is to build out the logic to assign these vehicles to the appropriate ferry and to handle when the ferry is full.

The goal of this test is to see understanding of:

  • SOLID programming practices
  • Understanding of basic UI practices
  • Create extensible code

Bonus POINTS

Each car type costs a different amount to board a ferry. The Ferry worker earns 10% of the total ticket cost per day.

Car Type Cost
Car $5
Van $7.50
Truck $10
Bus $15

For each vehicle that boards the ferry calculate how much:

  • the terminal earns
  • the worker earns
